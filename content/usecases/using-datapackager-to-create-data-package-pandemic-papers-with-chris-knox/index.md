[DataPackageR](https://docs.ropensci.org/DataPackageR/)

#### What did you do? 
Used DataPackageR to create a data package of the New Zealand Government's released [Covid-19 document dump](https://covid19.govt.nz/resources/key-documents-and-legislation/proactive-release/) available as searchable plain text.

Use case from **[from Chris Knox's tweet](https://twitter.com/vizowl/status/1265876658508054529)**


#### URL or code snippet for your use case
Output: [Pandemic Papers Data Package GitHub](https://github.com/nzherald/pandemicpapers)
