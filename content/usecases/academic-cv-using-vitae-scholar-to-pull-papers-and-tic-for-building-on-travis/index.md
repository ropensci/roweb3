#### What did you do? 

Academic CV built using R - updated automatically from [Google Scholar](https://scholar.google.com/citations?user=GqZm90IAAAAJ&hl=en) each week. See [here](https://www.samabbott.co.uk/cv/cv.pdf) for the full rendered CV. All other data stored as .csv's so can be updated on GitHub without touching the code or main document.
