This are scripts used for the Biodiversidata Project that are useful for:

1. Retrieving Conservation Status and Population Trend (IUCN).
2. Checking Species Names.
3. Retrieving Taxonomic Information for a Species.

#### Package or resource used
[rredlist](https://github.com/ropensci/rredlist), [taxize](https://github.com/ropensci/taxize)
