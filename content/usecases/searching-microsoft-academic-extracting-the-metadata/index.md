#### What did you do? 

My exercise comprise a piece of code that can help to search the articles and extract the metdata from Microsoft Academic. The example is limited with only one query type.
[microdemic](https://github.com/ropensci/microdemic)

[COVID March publications ](https://rpubs.com/alexeilutay/covid_coverage2)

The query consists of the words (terms) and the publication date range. Number of results were received via ma_calchist, the records were extracted by 100 with ma_evaluate.
The guy who learned me to test this was [Darrin Eide](https://twitter.com/DarrinEide)
