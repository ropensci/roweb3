#### What did you do?

This tutorial provides examples of queries of the OAI interface of Berlin State Library with R. It deals with exemplary queries addressed to the OAI-PMH interface of the Staatsbibliothek zu Berlin and also explains how to download the data and save it in a CSV file.

[https://lab.sbb.berlin/oai-interface-request-response-and-data-download-with-r/?lang=en](https://lab.sbb.berlin/oai-interface-request-response-and-data-download-with-r/?lang=en)

The tutorial includes a video (in German language only).
