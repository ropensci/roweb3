Nuestro Proceso de Revisión de Software

rOpenSci's Software Peer Review es un mecanismo para validar paquetes de alta calidad, mejorar las mejores prácticas y habilidades en la comunidad de software de investigación, y construir colaboraciones y la comunidad a través de un proceso de revisión transparente, constructivo y abierto utilizando la infraestructura de código abierto de GitHub . A través de ella hemos reunido un conjunto de cientos de herramientas para apoyar la ciencia abierta, y una comunidad de autores de paquetes, revisores y mantenedores para mantenerlos.
